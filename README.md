### For execute in your machine
<pre>

Install all dependencies
$ yarn install

On backend folder
$ yarn start

On frontend folder
$ yarn start

On mobile folder
$ yarn start

</pre>

The app mobile was built using <a href="https://expo.io/">Expo</a>
